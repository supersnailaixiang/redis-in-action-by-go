# 2 实现redis 结构

## 2.1 存储文章信息
 文章信息有多种所以需要用hash 存储。文章标题是hash名称
 eg：
 article:123456 {
     title
     link
     poster
     time
     votes
 }

 ## 2.2 存储排序功能（按照时间和分数）
 排序功能实用zset 
 acticle:1000 -> 分数或者时间

 ## 2.3 存储文章的投票用户

    user
